moogEvent

An Event structured to be read by the Incident.MOOG REST LAM
Type:
  • object
Properties:
Name Type Description
signature string the id used to correlate events into a single alert.
source string synonym for host, the main entity that is understood to be a logical component in the system.
source_id string an external identifier for the source entity.
external_id string an external identifier for the specific event (event UID).
manager string the logical name of the manager that transmits the event.
source string the actual component or sub component that produced the event.
class string an arbitrary classification for the event, to logically group it with other events of the same class.
agent_location string if the event is generated by an agent then the source entity that the agent process executes within.
type string an arbitrary type for the event, to logically group it with other events of the same type.
severity number the severity from 0-5, 0-lowest 5-highest.
description string the long form message from the event used as the human readable description.
first_occurred number the date and time the event first occurred, in epoc seconds.
agent_time string the date and time the agent identified the event, in epoc seconds.
